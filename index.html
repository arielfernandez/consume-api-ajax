Aerolab Coding Challenge 3

In https://aerolab.co/portfolio we show our latest shots in Dribbble using an internal API.

Find that endpoint and edit the dribbleShots function so it returns an array of shots. There's one extra challenge though: You need to read the tests to figure out how the shot objects should look like.

As an additional detail, the "image" key should be the hidpi image source. If the image does not have a hidpi source, we don't want it shown, so we remove it from the array.

Your challenge is to implement the dribbleShots function, making sure it passes all of the tests, which includes figuring out the correct format for the shots objects.
